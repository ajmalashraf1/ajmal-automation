**Upload instructions (one-shot):**
1. Create a new GitHub repo: https://github.com/new
   - Name: ajmal-automation
   - Public
   - Do NOT initialize with README

2. On repo page, click "Add file" → "Upload files"
   - Drag & drop ALL files and folders from this package (index.html, landing_page.html, branding/)
   - Commit changes

3. Enable GitHub Pages:
   - Settings → Pages → Branch: main, Folder: / (root), Save
   - Wait ~30-60s

4. Replace placeholders in index.html:
   - mailto:ajmal@example.com → your email
   - wa.me/91XXXXXXXXXX → your WhatsApp number (no +)

If you prefer to upload via ZIP:
- Upload the ZIP contents (not the ZIP file) via "Upload files" or use the desktop git commands.
